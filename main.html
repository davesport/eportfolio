Welcome!

This is my e-portfolio site, published (privately) on Github, for free. I made this site using HTML, CSS, and Javascript, which I learned in three web development courses in SJSU's MLIS program. I wrote my HTML and CSS on Visual Studio Code, a free computer app.


Navigation:

The navigation bar at the top of the page contains links to each page of my portfolio. This bar is in the same place on each page.

Within each page, there is a sidebar to the left (same as this sidebar) that defines and outlines each competency. Click the quick links there to jump to a particular part of a competency.

My contact information is listed at the bottom of each page.

This website has a responsive layout to different screen widths, which means that the layout of each page will change as your screen or browser width changes (in order to better suit your screen size).


Contents:

My portfolio contains an introduction, conclusion, and competencies A-O, which break down what I have learned and practiced in this program into thematic groups.
